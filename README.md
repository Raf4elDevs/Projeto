## Descrição

Este projeto consiste no desenvolvimento de um robô assistente físico com inteligência artificial, projetado para auxiliar usuários durante o preparo de receitas culinárias de forma prática, intuitiva e personalizada.
O sistema atua como um guia interativo na cozinha, oferecendo instruções em tempo real, adaptadas ao nível de conhecimento do usuário desde iniciantes até cozinheiros mais experientes. A proposta é transformar a experiência de cozinhar em algo mais acessível, eficiente e agradável, reduzindo erros e aumentando a confiança do usuário durante o processo.
